PROJECT NAME: Accounting Ledger Application

What does it do? Allows users the ability to quickly access past and new transactions from their ledger. 
They are also able to make payments and deposits.

HOME SCREEN
This is a simple and straight forward home screen that allows the user to choose between 4 options: 
Add deposit, Make a payment, Access the Ledger and exit.

LEDGER SCREEN
When a user selects "L" for Ledger, the LedgerScreen is then accessed. The user has these features to display: 
All transactions, Deposits, Payments, Reports and to return to the HomeScreen.
